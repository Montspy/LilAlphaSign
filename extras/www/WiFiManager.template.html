How to trigger alerts

Black is required. Blue and Green parameters are optional
  • Sweep
    • http://{ip}/alert/sweep?c=0xff00ff&s=100
    c: color [hex RGB color]
    s: speed [10~500%]
  • Fill
    • http://{ip}/alert/fill?c=0xff0000&s=100
    c: color [hex RGB color]
    s: speed [10~500%]
  • Flash
    • http://{ip}/alert/flash?c=0xffff00&s=100
    c: color [hex RGB color]
    s: speed [10~500%]
  • YouTube Intro
    • http://{ip}/alert/intro?c=0xffffea&g=false
    c: color [hex RGB color]
    g: add glitches [true/false]

How to change the background

  • Rainbow
    • http://{ip}/bg/rainbow&s=100
    s: speed [10~500%]
  • Cycle hues
    • http://{ip}/bg/cycle&s=100
    s: speed [10~500%]
  • Off
    • http://{ip}/bg/off
    Alerts will still show
  • Solid color
    • http://{ip}/bg/color?c=0xffffea
    c: color [hex RGB color]
  • Heartbeat
    • http://{ip}/bg/beat?c=0xffffea?s=100
    c: color [hex RGB color]
    s: speed [10~500%]

Settings and misc

  • Settings
  • Disable LEDs (background AND alerts)
    • http://{ip}/off
  • Enable LEDs (background AND alerts)
    • http://{ip}/on
